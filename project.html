The big day -- v0.3 available!

New advancements to the development of UFC have been made. The third and last version of the game is now available! Click here to go to the download page. The changelog is as follows:

(+) Networking
    (→) Fixed various inconsistencies in the networking.

(+) User Interface
    (→) Added the options menu (contains a volume slider).
    (→) Added the pause menu.
    (→) Added the lobby menu, where players can create or join a match.

(+) Sound and music
    (→) Added a new music track that plays in the lobby menu.

(+) Players
    (→) The characters are now animated and fully modeled / textured.
        (→) Punch animation
        (→) Kick animation
        (→) Die animation
        (→) Idle animation
        (→) Sidestep animation
    (→) The fighters can now do the protection move.
        (→) Reduces damage taken, but player is immobilized.
(→) The fighters can now perform combos.
        (→) K - K - M (punch, punch, fire) throws an ice ball.
        (→) D - A - V : strength up (next 5 attacks powered up)
        (→) C - H - A : jump up (triple jump enabled)
        (→) P - H - I : speed up (more speed for a short distance)
    (→) The fire balls now produce beautiful particles.

See you next year!

Newsletter 4 -- v0.2 available!

New advancements to the development of UFC have been made. The second version of the game is now available! Click here to go to the download page. The changelog is as follows:

(+) Networking
    (→) Fixed health bars not synchronized across the network.
    (→) Fixed desynchronizations in movements when playing on a network.
    (→) Fixed various inconsistencies in the networking.

(+) User Interface
    (→) Main menu heavily modified.
        (→) Beautiful backdrop added.
        (→) Each button is part of the backdrop.
        (→) Added the options button.
    (→) Stamina bar added.

(+) Sound and music
    (→) Added dozens of sound effects and voices for various player actions.
        (→) Hit sound effects.
        (→) Death sound effects.
        (→) Fire ball sound effects.
        (→) Jump sound effects.
        (→) Sound effects for each button in the main menu.
    (→) Added a new music track that plays during fights.

(+) Players
    (→) The fighters can now double jump.
    (→) The fighters now have a stamina bar.
        (→) Each action from a fighter reduces their stamina.
        (→) The stamina bar slowly refills over time.
    (→) The fighters can now dash forward (or backward).
    (→) The fighters can now throw fire balls!
